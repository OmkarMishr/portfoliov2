# Neue Machina fonts

The site is wired to use **Neue Machina** as the primary display/body font.
Drop the free `.woff2` files here so the exact font goes live:

- `NeueMachina-Light.woff2`
- `NeueMachina-Regular.woff2`
- `NeueMachina-Ultrabold.woff2`

Get them from Pangram Pangram Foundry (free for personal use):
https://pangrampangram.com/products/neue-machina

If the originals are `.otf`, convert to `.woff2` (e.g. with `fonttools` or an
online converter) and keep the exact filenames above.

Until these files exist, the site falls back to **Space Grotesk** (a close,
fully-licensed geometric stand-in loaded via `next/font`).
